About

I am a graphics and web designer, my passion for what i do as a web designer has motivated me in setting up this little hints on
how to be a web designer. The main goal of this website is to help you on your journey on how to be a web designer.

Important steps

Web design and development is an amazing skill which makes you a web designer or a web developer, with best practices you could find yourself designing amazing world class websites.
When learning or practicing web design, it is a good choice to make us of Google or Youtube to search for tutorial websites or videos to guide you in practicing. Some of the tutorial websites you could make use of are: W3School, Lynda and Udemy.

Follow best practices

The first step in learning web design and development is following best practices from other web designers or websites. Studying their layout, menu and context area of the website.

Learn the fundamental
languages and programs

HTML, CSS, Javascript, JQuery, PHP, MySQL, Bootstrap. etc.

To start right on, the first three languages which makes it easier for you to learn web design and master coding are: HTML, CSS and Javascript. Be a master of this first three then move on to learn the rest of them; jQuery, PHP, MySQL

Making use of dynamic
code editors

There are several code editors which you can use to start right on to learn web design, knowing how to code is a good quality skill for you to learn. Some of the few code editors you can use are: Notepad++, Sublime text, Textwrangler.etc.

Practice Often

Get yourself busy after learning all these skills. Constant practice makes you master them. So get coding!
