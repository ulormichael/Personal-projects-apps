What you need to know to learn web design

The first to take in learning any new skill is "Getting started". So, to learn web design you need learn the following mark up language and programming languages.eg. HTML, CSS and JAVASCRIPT.

(1) HTML: HTML is a markup language (HTML meaning Hypertext Markup Language) which is actually used to create the content of the web page.

(2) CSS: meaning (Cascade Styling Sheets) is actually responsible for the design and style of the actual website. CSS is used to control and style the layout of the web page, control presentation, give background color of the web page, formatting it and also given it a visual effect.

(3) Javascript (JS): is a dynamic programming or scripting language used to control the behaviour of different elements. Javascript is to define the interactive elements of a web page which help to engage end users.
